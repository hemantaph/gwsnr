gwsnr
======

``gwsnr``: Gravitational Wave Signal-to-Noise Ratio Computation Package

The ``gwsnr`` package is designed to facilitate efficient and accurate SNR computations in gravitational wave research. It implements advanced techniques for enhancing calculation speed and precision, making it a valuable tool for researchers in this field. Description available in `README.md <https://github.com/hemantaph/gwsnr>`_

The code is available at `github <https://github.com/hemantaph/gwsnr>`_
